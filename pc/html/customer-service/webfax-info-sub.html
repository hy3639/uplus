U+웹팩스는 업무방식에 따라 내부 시스템에 연동하여 자동/대량
발송하거나, 웹/모바일 및 문서만 열어 바로 발송 가능합니다.

  • 웹방식 : 웹에서 팩스 수/발신 및
    알림 설정
    • 대상 : 개인(선불), 법인/개인사업자(후불)
    • 메일과 유사한 팩스 수/발신 화면
    • 수/발신 시 다양한 알림 (이메일 / 문자 / 카톡 / 팩스)
    • 붙여넣기로 쉬운 수신자 추가 및 주소록 관리
    • 1인 1팩스를 위한 아이디 무제한 생성 (사업자/후불)
    개인(선불)신청 사업자(후불)신청
  • 호스트 방식 : 웹팩스 API를 내부
    시스템
    연동하여 자동 대량 발송
    • 대상: 법인/개인사업자(후불)
    • 전송 파일 동시 변환으로 대량 자동 발송
    • 내부 ERP, 그룹웨어 등 연동
    • JAVA 기반으로 다양한 OS./DB지원
    모듈 신청 메뉴얼 다운로드
  • 모바일 방식 : 외부에서 팩스 수/발신 및
    알림
    • 사진 촬영하여 팩스 발송
    • 휴대폰, 신용카드로 선불 충전 후 즉시 사용
    • 웹과 싱크 및 동일 기능 사용
    • 모바일로 간편하게 서비스 신청(선불)
    모바일앱 다운로드(Google Play 스토어)
  • 간편 팩스 방식 : 문서에서 바로 팩스
    발송이
    필요할 때
    • 홈페이지 접속 없이 프린트 버튼으로 발송
    • 웹주소록 연동으로 발신자 선택 발송
    • 팩스 수신 알림로 즉시 확인 가능
    • 간편한 프로그램 설치로 즉시 이용 가능
    간편팩스 다운로드 사용가이드

업무의 시간을 줄여주는 편리한 기능

U+웹팩스는 팩스 수발신을 편리하게 이용/관리
할 수 있도록 다양한 기능을 제공합니다.

  • 각 기능에 대한 상세 내용은 팁팝업 오픈 동영상팝업 오픈 을 누르시면 확인하실 수 있습니다.

    (수신 필요 시)

    - 매월 1일 6,600원 이상 충전되어 있지 않을 경우 수신 서비스가 자동 중지되며, 충전 후에는 내정보관리 > 수신번호 관리에서 "수신번호 재사용"을 클릭해야 정상 수신 가능.
    - 2개월 동안 충전하지 않고, "수신번호 재사용"을 클릭하지 않을 경우 수신번호 자동 해지.
