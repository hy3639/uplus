Return Fax 란?
  • 기능 : 팩스 수신 시 자동으로 상대방에게 수신완료 내용을 FAX와 문자로 전달하는 부가서비스 입니다.
  • 메시지 변경 : 고객센터로 문의하시기 바랍니다.
  • 타입변경 : 리턴 FAX 해지 후 다시 신청하시기 바랍니다
  • 요금 : 장당 42.9원(부가세 포함)으로 청구됩니다.
타입선택 및 신청
  • 모든 수신팩스 적용: 수신되는 모든 팩스에 대해서 Return FAX 를 발송하고 싶은 경우.
  • 주소록 팩스번호만 적용: 주소록에 있는 팩스 번호에서 송신된 팩스에 대해서 Return FAX 를 발송하고 싶은 경우.
Return FAX 신청(모든 수신팩스 적용) Return FAX 신청(주소록 팩스번호만 적용)
Return SMS 란?
  • 팩스 수신 시 자동으로 상대방에게 수신완료 내용을 SMS로 전달하는 부가서비스 입니다.
  • 발송 불가 사례 :
    • ① 주소록에 저장되어 있지 않은 팩스 번호로 수신
    • ② 매칭되는 핸드폰 번호가 없을 경우
    • ③ 원하는 팩스번호만 선택하여 SMS 발송
  • 요금 : 건당19.8원(부가세 포함)로 청구됩니다.

※ 발송 예시

  • 주소록에 저장된 번호가 아래와 같을경우
    • 이름: LG U+
    • 팩스번호: 02-1234-5678
    • 핸드폰: 010-1111-2222 ▶ '02-1234-5678' 로 부터 팩스 수신시, 자동으로 등록되어 있던 핸드폰번호 '010-1111-2222'로 Return SMS가 발송됩니다.
