수신번호/주소록 추가 영상 가이드

  • 팩스 수신 및 받은팩스함은 회원가입 시 또는 [수신번호 신청/관리]에서 수신번호 신청 후 이용이
    가능합니다.
  • 최초 가입시에는 발신전용 번호만 부여되며, 수신번호를 별도로 신청 시 해당번호는 발신/수신이
    동시에 가능한 수신번호 입니다.
  • 수신번호를 신청 시 최저 이용료 3,000은 면제됩니다. (후불 고객)
  • 0503 번호는 선불고객 전용 번호입니다.

1. 번호 선택

  • ① 지역번호 및 국번을 선택하면 하단에 신청 가능한 번호가 노출됩니다.
  • ② 해당번호 선택 후 [신청하기]를 누르시면 신청이 됩니다.
  • ③ 신청된 번호는 하단 표에 [신청]이라고 노출됩니다.

2. 수신번호 관리

  • ① 수신번호 신청/관리의 수신번호 현황에서 선택한 수신 번호를 대표번호로 설정할 수 있습니다.
  • ② 서비스 해지 시 수신 번호 이용이 중단되며, 해당 번호로 팩스를 받을 수 없습니다.

선불 고객 필수 확인

  • 선불 고객은 수신번호 신청 시 아래 내용을 꼭 확인하시기 바랍니다.
일반번호/
070
  • 수신번호를 신청하면 매월 1일에 충전금액에서 이용료 6,000원(VAT 미포함)이 차감됩니다.
    • ※ 매월 1일 잔여금액이 6,000원 이하일 경우 수신번호 상태가 자동으로 변경되므로
      반드시 충전해 주셔야 합니다.
  • 이미 "일시정지"가 된 경우, 해당 금액을 충전하고 내정보관리 > 수신번호관리에서
    "수신번호 재사용"을 클릭해야 정상 수신이 가능합니다.
  • "일시정지"상태에서 남아있는 수신번호 사용 일 수가 잔액의 범위 안에 있는 경우
    (1일 200원으로 계산) 자동 차감되면서 정상 상태로 복구됩니다.
  • 2회차 때 충전하지 않고, "수신번호 재사용을" 클릭하지 않을 경우 수신번호가 자동
    해지되오니 양해 부탁드립니다.
    예 ) 5/15일 가입 후 수신번호 신청
    • 6/1일(1회차) - 충전금액이 6,000원 이하일 경우 -> 번호 일시정지
    • 6/6일 - 충전금액이 5,000원 남아 있는 경우 -> 5,000원 차감되면서 정상 상태로 복구
    • 7/1일(1회차) - 충전금액이 6,000원 이하일 경우 -> 번호 일시정지
    • 8/1일(2회차) - 충전금액이 200원 이하일 경우 -> 번호 해지
    해지된 번호는 재사용하실 수 없으니 이점 양해 부탁드립니다.
0503번호
  • 수신번호를 신청하면 매월 1일에 충전금액에서 이용료 1,000원이 차감됩니다.
    • ※ 매월 1일 잔여금액이 1,000원 이하일 경우 수신번호 상태가 자동으로 변경되므로 반드시 충전해 주셔야 합니다.
  • 1회차: 일시중지 / ㆍ2회차: 해지
  • 해지된 번호는 재사용하실 수 없습니다.
  • 한 번 사용한 번호 이용료는 환불 불가합니다.

1. 주소록에서 추가 방법

  • ① 직접 입력
    • 주소록 입력창에서 직접입력 또는 Local 문서에서 복사하여 추가하실 수 있습니다.
    • [새행 추가] 버튼으로 행을 추가하여 주소를 입력하실 수 있습니다.
    • 해당 주소를 삭제할 경우 체크박스에 체크를 한 후 [행삭제] 버튼으로 삭제합니다.
    • 그룹명을 선택하고, [저장]을 클릭하여 완료합니다.
    • 그룹명을 지정하지 않을 시에는 그룹없음을 선택합니다.
  • ② 엑셀 불러오기
    • 엑셀 불러오기 버튼을 클릭하여 [샘플파일 다운로드]하여 서식을 완성합니다.
    • 반드시 아래 예시와 같은 순서로 작성되어야 하며, 텍스트 형식으로 입력해주시기 바랍니다.
    • 대표번호를 등록하실 경우 해당 지역번호까지 입력해주셔야 합니다.
      (예 : 대표번호가 1544-8007(서울)일 경우 02-1544-8007 입력)
    • [파일 첨부]를 클릭하여 해당 서식파일을 첨부합니다.

2. 팩스 보내기에서 추가

수신번호 입력 후 체크박스에 체크 후 [주소록 추가] 버튼으로 주소록에 추가할 수 있습니다.

3. 받은팩스함에서 추가

주소록에 등록되어 있지 않다면 발신번호에 마우스커서를 올려 [주소록 추가] 버튼으로
주소록에 바로 등록할 수 있습니다.

등록 후에는 보낸사람 항목에 이름과 해드폰 번호가 노출됩니다.

4. 보낸팩스함에서 추가

주소록에 등록되어 있지 않다면 수신번호에 마우스커서를 올려 [주소록 추가] 버튼으로
주소록에 바로 등록할 수 있습니다

등록 후에는 받는사람 항목에 이름과 해드폰 번호가 노출됩니다.
