SMS 인증 접속

설정 시, 전체 사용자 대하여 적용되므로, 설정 전 아이디관리에서 서브아이디별 핸드폰번호 유무를 확인하세요.
- 설정 후 서브아이디별 해제는 아이디관리에서 가능합니다.
- 서브아이디당 1개의 핸드폰으로 인증이 가능하므로 다수가 공유하는 경우 개별 아이디를 생성하여 적용하시기 바랍니다.
적용 설정 후, 인증프로세스를 적용할 부문을 1개이상 선택 필수 (로그인 / 팩스발송 / 문자발송 / 회원정보변경 / 아이디관리)
- 단, U+웍스로부터 SSO로 접속될 경우에는 웹팩스로 자동로그인이 되기 때문에 로그인 시 SMS 인증은 적용되지 않습니다.
