개인정보 발송 차단

팩스 발송 시, 발송 팩스 문서에 대인정보가 포함되어 있을 경우
자동으로 발송을 차단합니다.
(후불 관리자회원만 신청 가능합니다.)
2016.11.18 / 서비스사용중

서비스 설명

  • 발송 팩스 내용에 개인정보(주민등록번호, 휴대폰번호, 신용카드번호, 이메일 주소)가 있을 경우 자동으로 발송 차단을 하는 부가서비스로,
    이미지/텍스트 모두 구분됩니다.
  • 하이픈'-'이 없는 개인정보는 발송 시 차단이 되지 않습니다.

사용방법 안내

  • 1) 서비스 신청
    • U+웹팩스 홈페이지 [관리자 > 부가서비스신청 > 개인정보 발송 차단] 화면의 서비스신청하기 버튼을 통하여 온라인 상에서 실시간 가입이 가능합니다.
  • 2) 서비스 해지
    • U+웹팩스 홈페이지 [관리자 > 부가서비스신청 > 개인정보 발송 차단]화면의 해지 버튼을 통하여 서비스 해지가 가능합니다.
